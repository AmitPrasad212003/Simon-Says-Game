🎮 Simon Says

Press any key to start the game

📌 How to Play

  1. Press any key to start the game.
  2. Watch the colored button that flashes.
  3. Click the same button.
  4. Each round, a new button will be added to the sequence.
  5. Repeat the full sequence in the correct order.
  6. If you make a mistake → ❌ Game Over.
